# Projet d'ouverture informatique

## Description

Ce projet comprend:

- un jeu de platforme en 2D réalisé avec pygame.
- un éditeur de niveau réalisé avec pygame.
- des diagrammes [UML](./uml/) pour mieux comprendre l'architecture du projet.
